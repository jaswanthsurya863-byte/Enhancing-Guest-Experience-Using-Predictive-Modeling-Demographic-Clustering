Enhancing Guest Experience Using Predictive Modeling & Demographic Clustering

This project applies machine learning and clustering techniques to hotel booking data to predict guest special requests at the time of booking and segment customers based on demographics. The goal is to help hotels improve guest experience, service personalization, and resource planning.

📌 Project Overview

Dataset: 120,000+ real hotel booking records (Kaggle)

Hotels often struggle to anticipate guest needs in advance. This project builds predictive models to identify whether a guest is likely to make special requests and uses demographic clustering to uncover meaningful customer segments.

🔍 Key Highlights

Performed data cleaning, preprocessing, and exploratory data analysis (EDA) on booking and demographic data.

Engineered a binary target feature to predict whether a guest will place a special request.

Built and evaluated multiple models:

Decision Tree

Logistic Regression

Random Forest (best model)

Random Forest Results:

Accuracy: ~82%

AUC: ~0.88

Conducted feature importance analysis using:

Information Gain

Gain Ratio

Random Forest importance

Applied KMeans (k = 3) and DBSCAN for demographic clustering.

Visualized clusters using PCA for dimensionality reduction.

👥 Guest Segments Identified

Planned Domestic Guests

Budget Domestic Travelers

Luxury Foreign Guests

These segments help hotels better understand guest behavior and preferences.
